Maildir
=====

Store email/backup in maildir format.

Install
====
- `$git clone git://github.com/kracekumar/maildir.git`
- `$cd maildir`
- `$python setup.py develop`
- Requires SQLITE3 installed.

Usage:
===
- `$ maildir -h`

- `$maildir -config /path/to/config.py`
- `$cat maildir/sample-config.py`
